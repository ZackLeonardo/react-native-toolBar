# react-native-toolbar

## Installation
The component which can be used as topToolBar and bottomToolBar.

## Usage
 (for myself in private npm store)
```
yarn add @zdy/react-native-toolbar
```
(for everyone)
```
yarn add file:/...path.../react-native-toolbar
